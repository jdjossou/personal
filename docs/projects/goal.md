4. Projects — Quest Screen
This is your project showcase.
It should feel like a Persona quest log, not a generic portfolio card grid.
Include:
Numbered project rows like /01, /02, /03
Project name
Date range
Status badge: SHIPPED, IN PROGRESS, or ARCHIVED
Tags
Expandable or selectable project details
Links to GitHub, demo, or related material
Screenshots only when useful
The important rule: no generic rounded project cards. This section should feel like a mission list.
Please refer to the `docs/projects/reference.png` reference image.
It should be easy for me to add projects in the future.
It should also be possible to send the link of the page that opens directly the project page.
