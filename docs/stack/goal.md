The goal of this section is to build the stack section of the website. Like all other sections, this section is based on a Persona 3 Reload menu. More precisely, it is based on the menu shown in the reference `docs/stack/stack_reference.png`. The end goal is to have a page to list the technologies I know in different categories. More specifically, I want, as a start, to show the following technologies below:
Languages: Java, Python, C++, TypeScript, JavaScript, Swift, HTML/CSS, Dart
Frameworks & Libraries: Spring Boot, React, Next.js, Angular, PyTorch, NumPy, FastAPI, Flutter
Tools & Platforms: Git, Kubernetes, Kafka, Jenkins, Dynatrace, Postman, Jira, 
Certifications: Microsoft Azure AZ-900, Microsoft Azure AI-900
Databases: PostgreSQL, Oracle SQL, MongoDB, Firebase Firestore
Spoken languages: English, French

The goal is also to reproduce close to the reference. However, there are few things we won't include. Those are:
- The character in the background
- The leader and characters sprites on the left. The characters on the left should be replaced by the categories. No level, health or mana stats.
- The cost and sp cost on the right list. The elements in that list will be the skills from the current category. There will be a leading icon for each category.
