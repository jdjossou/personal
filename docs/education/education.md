5. Education — Stats Screen
This section presents your academic profile like a character stats screen.
Include:
University name
Degree / program
Expected graduation date
Relevant courses
Course list styled like abilities or learned skills
No GPA
The vibe should be factual and clean, almost like a system profile.
Please refer to the `docs/education/education_reference.png` reference image. Please use the reference as the standard for what the page should look like visually.
