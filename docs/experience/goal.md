6. Experience — Social Link Screen
This is your work experience section.
Instead of résumé bullets, it should feel like journal entries for each role.
Include:
Company name
Role title
Date range
A short journal-style paragraph about what you built or contributed
Most recent role first
Company/system-like visual treatment
The key difference from a normal portfolio: this should not read like “responsible for X.” It should read like “I built X, which helped Y.”

This page should feel unique, like all the other screens. To do this, use the `docs/experience/experience_reference.png` as a reference. Use the reference to adapt the Persona screen to the content we want to include in the experience page.
